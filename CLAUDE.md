# GeometryEngine

## 项目概述
本项目是一个基于 C++17 实现的二维几何引擎，重点关注：

- 计算几何算法
- 几何鲁棒性（数值稳定）
- 高性能空间计算

---

## 项目结构

### 源码目录
  - include 头文件目录
  - src 源文件目录
### 模块
  - base 基础类型
  - geometry 几何类型
  - algorithm 算法
  - dataengine 数据引擎
  - app 可执行程序入口
  - tests 单元测试
## 核心架构

### 几何类型体系
- Geometry（抽象基类）
- Point:表示一个点
- Line:表示折线
- Region:表示多边形
### 数据引擎
- IReader(读取为几何数组的基类)
- WktFileReader:读取wkt文本文件为几何数组的类，文件每一行是一个wkt串
## AI 行为规范（关键）

当 AI（Claude）参与开发时：
### 代码生成
- 生成的函数必须添加注释，如果是类成员函数，只需要在类中函数定义处添加注释
示例
```
//! \brief 获取指定索引处的顶点坐标（只读引用）
//! \param i 获取指定索引处的顶点坐标（只读引用）
//! \return 指向对应顶点的常量引用
const Coord &vertex(std::size_t i) const override;
```
- 生成的函数必须有对函数逻辑流程的注释
- 头文件只允许声明，不允许实现（除 inline）
- 所有实现必须放在 `.cpp` 文件中
- 禁止在类型内部实现复杂算法
- 子类必须重写所有基类的纯虚函数
- 各种类型命名规范和现有代码保持一致
- double数值相关的比较或者计算算法必须引入EPS
